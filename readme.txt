== Description ==

Let you add a customizeable widget to display the news from http://www.aceto-balsamico.net/ in your sidebar.

Features:
- widgeht is customizeable
- you can define how much postings are displayed
- you can short the title 

== Installation ==

1. Extract `balsamico-news.zip`.
2. Upload the folder `balsamico-news` to the `/wp-content/plugins/` path.
3. Activate the plugin through the 'Plugins' menu in WordPress Adminpanel.
4. Add the widget to your sidebar from Appearance -> Widgets and configure the widget options.
